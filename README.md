# 🛒 E-Commerce Website

A fully responsive online store built with **Next.js**, **Tailwind CSS**, and **REST APIs** — featuring a product catalog, shopping cart, checkout flow, and real-time data fetching.

🔗 **Live Demo:** [bmaged510-cell.github.io/E-commerce](https://bmaged510-cell.github.io/E-commerce/)

---

## ✨ Features

- 🛍️ Product catalog with 50+ products
- 🛒 Shopping cart with add / remove / update quantity
- 💳 Checkout flow with order summary
- ⚡ Real-time data fetching via REST APIs — sub-2s load time
- 📱 Fully responsive — mobile, tablet, and desktop
- ♻️ Reusable component architecture
- 🔁 Efficient state management using React Hooks

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| **Next.js 14** | Framework & routing |
| **TypeScript** | Type safety |
| **Tailwind CSS** | Styling |
| **REST APIs** | Product data fetching |
| **React Hooks** | State management (useState, useEffect, useContext) |
| **Axios / Fetch** | HTTP requests |

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/bmaged510-cell/E-commerce.git

# Navigate to the project
cd E-commerce

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
